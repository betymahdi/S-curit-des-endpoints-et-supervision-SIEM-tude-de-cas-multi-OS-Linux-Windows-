# S-curit-des-endpoints-et-supervision-SIEM-tude-de-cas-multi-OS-Linux-Windows-
Plateforme de supervision et de sécurité des endpoints Windows et Linux basée sur Wazuh, combinant SIEM et EDR. Le projet permet la collecte, l’analyse des logs, la détection des menaces et la génération d’alertes dans un environnement multi-OS sur AWS.
